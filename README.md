# HTML • CSS • Bootstrap • JavaScript Öğrenme Notları ve Alıştırmalar

Bu repo; **HTML, CSS, Bootstrap ve JavaScript** öğrenirken yazdığım **notları**, **küçük örnekleri** ve **pratik alıştırmaları** bir arada tutmak için oluşturuldu.  
Amaç; konuları parça parça öğrenip, örnek kodlarla pekiştirmek ve gerektiğinde hızlıca geri dönüp bakabilmek.

## İçerik
- **HTML**: temel etiketler, formlar, semantik yapı
- **CSS**: seçiciler, box model, flexbox/grid, responsive tasarım
- **Bootstrap**: grid sistemi, bileşenler, yardımcı sınıflar
- **JavaScript**: değişkenler, fonksiyonlar, DOM, events, basit projeler

> Not: Dosyalar “öğrenme süreci” kapsamında olduğu için bazı örnekler farklı yaklaşımlarla tekrar edebilir.

## Nasıl kullanılır?
- Kodu incelemek için dosyaları doğrudan açabilirsin.
- Çalıştırmak için çoğu örnekte `.html` dosyasını tarayıcıda açman yeterli.
- Bazı örnekler harici CDN kullanıyor olabilir (Bootstrap gibi).

## Klasör Yapısı (örnek)
Aşağıdaki yapı repoda birebir aynı olmak zorunda değil; repo büyüdükçe düzenlemek için referans olarak düşünebilirsin:

- `html/` → HTML örnekleri
- `css/` → CSS örnekleri
- `bootstrap/` → Bootstrap örnekleri
- `javascript/` → JS örnekleri
- `projects/` → mini projeler / sayfa çalışmaları

## Hedefler
